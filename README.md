# UPath – Educational Platform Prototype

UPath is a frontend prototype of an educational platform designed for students aged 7–20.
The project focuses on safe tutor discovery, short-note learning, emotional feedback,
and career path guidance.

## Features
- Tutor discovery interface
- Short-note learning UI
- Career path guidance form
- Emotional feedback collection
- Student / Tutor authentication UI

## Tech Stack
- HTML5
- CSS3
- Responsive Web Design

## Project Status
This project is a frontend prototype.
Backend services and AI-based recommendation systems are planned for future development.
